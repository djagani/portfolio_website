Project information

  • Category: UI/UX Internship Work

My role primarily involved designing and developing user interfaces for the company's products, utilizing frameworks such as Angular 14, Bootstrap, Figma and development tools such as HTML, CSS, JS.

Dashboard UI

The design is characterized by a cool-tone color scheme, fostering a visually calming and professional atmosphere. Integrating diverse chart types—such as pie charts, line charts, radial charts, donut charts and tree map—the dashboard ensures comprehensive data visualization for enhanced user insights. The donut chart leads the user's eyes around the arcs of the circle and tends to create more of a narrative. The line graphs show how data changes over time and are used to compare different data sets.

Login Page

In crafting the Login Page UI, I seamlessly blended a cool-tone color scheme with a cybersecurity theme, embodying both sophistication and security. The minimal design ensures a streamlined user experience, prioritizing simplicity and functionality. This project reflects my ability to merge aesthetic considerations with the specific requirements of a cybersecurity-focused interface, offering a visually pleasing yet secure entry point for users.
